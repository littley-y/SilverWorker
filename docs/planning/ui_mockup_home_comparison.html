수정 사항 요약
1️⃣ 마스코트를 최상단 배너로 이동 (AppBar 제거 → 스크롤 시 함께 내려감). 강아지 애니메이션 + 랜덤 응원 메시지.
2️⃣ 필터를 가로 스크롤(슬라이드) 칩 형태로 변경.
3️⃣ 카드 구조 변경: 최상단 업무강도 토큰(좌) + 급여(우) → 제목(큰 폰트) → 회사명(얇은 폰트) → 하단 연회색 토큰 3개(구직형태·도보거리·D-day) 한 줄 배치.
4️⃣ 제목에서 "모집" 제거.
5️⃣ 업무 강도를 문장 표현 + 테두리 pill 스타일. 색상: 초록(하) / 주황(중) / 빨강(상).

## 3. 구현 상세

### 3.1 visibleJobListProvider

```dart
final visibleJobListProvider = FutureProvider<List<JobModel>>((ref) async {
  final jobs = await ref.watch(jobListProvider.future);
  final user = ref.watch(authStateProvider).valueOrNull;
  if (user == null) return jobs;

  try {
    final applications = await ref.watch(myApplicationsProvider(user.uid).future);
    final appliedIds = applications.map((a) => a.jobId).toSet();
    return jobs.where((job) => !appliedIds.contains(job.jobId)).toList();
  } on Exception {
    return jobs; // 필터링 실패 시 전체 공고 표시
  }
});
```

- `authStateProvider`로 현재 사용자 확인
- `myApplicationsProvider`로 지원 내역 조회 (이미 `cancelled` 제외됨)
- 지원한 `jobId` Set을 만들어 필터링
- 예외 발생 시 전체 공고 표시 (graceful degradation)

### 3.2 ApplicationRepository.fetchApplications 필터

```dart
.where('status', isNotEqualTo: 'cancelled')
.orderBy('status')
.orderBy('submittedAt', descending: true)
```

- `isNotEqualTo: 'cancelled'`로 취소된 지원 제외
- 복합 인덱스 `status + submittedAt` 필요 (기존에 이미 배포됨)

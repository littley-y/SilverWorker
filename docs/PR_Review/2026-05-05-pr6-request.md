## 구현 상세

### ApplicationFormScreen
- **공고 요약 카드**: `Container` + `AppColors.background`, 제목(title) + 회사명(caption)
- **자기소개**: `TextField` maxLength 200, maxLines 5, hintText "간단한 자기소개를 작성해 주세요 (선택)"
- **오터치 방어**: `_isSubmitting` → 버튼 disabled + spinner, 1.5초 후 해제
- **에러 처리**: already_applied → 버튼 "이미 지원한 공고입니다" + 비활성화, job_closed → SnackBar, 기타 → "다시 시도" SnackBar
- **하단 버튼**: `SafeArea` + 56dp, `bottomNavigationBar` 사용

### ApplicationRepository.submitApplication
- `FirebaseAuth.instance.currentUser!.uid` 로 사용자 식별
- `where('jobId', isEqualTo: jobId)` 로 중복 체크
- `isActive` / `deadline` 로 마감 체크
- Denormalized: `jobTitle`, `companyName` 함께 저장
- `FieldValue.serverTimestamp()` 사용

## 리뷰 포인트

1. **JobModel getter 추가**: `employmentTypeLabel`, `physicalIntensityLabel`, `physicalIntensityColor` — getter 위치가 모델 계층에 적절한지
2. **AppRoutes 헬퍼**: `jobDetailRoute()` 등의 파라미터화된 경로 빌더가 go_router 패턴과 충돌 없는지
3. **JobRepository DI**: 생성자 주입 변경이 기존 테스트(있다면)에 영향을 주지 않는지 — 현재 JobRepository 테스트는 없음
4. **Logger 사용**: `logger` 패키지 v2의 API(`error:` named parameter)가 올바르게 사용되었는지
5. **PrimaryButton / SnackBar 유틸**: 신규 파일의 구조가 프로젝트 컨벤션(`abstract final class` 등)과 일치하는지
6. **삭제 파일**: `badge_widget.dart`, `loading_overlay.dart` 삭제가 테스트나 import에 영향을 주지 않는지 (flutter analyze / flutter test 모두 통과 확인됨)

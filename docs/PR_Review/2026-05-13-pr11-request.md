## 1. 개요

홈 화면(JobListScreen) 및 공고 카드(JobCard)의 UI/UX를 개선합니다.
시니어 사용자의 가독성과 정보 밀도를 높이고, 마스코트를 통한 감성 요소를 추가합니다.

- **마스코트 배너**: 화면 최상단 이동, 터치 시 랜덤 응원 메시지, 강아지 캐릭터
- **공고 카드 재설계**: 한 화면 2개 노출, 업무 강도 문장 표현, 하단 메타 chip 통일
- **필터바 슬라이드**: 가로 스크롤 칩 형태로 변경

## 3. 구현 상세

### 3.1 MascotBanner

- 위치: AppBar 아래, FilterBar 위. 스크롤 시 함께 날라감 (고정 아님)
- 배경: `primaryLight` → white 그라데이션
- 왼쪽: 마스코트 이미지 72dp, 터치 시 랜덤 메시지 변경 + 햅틱
- 오른쪽: 말풍선 (흰색 배경, primary 테두리, 좌하단 꼬리)
- 메시지 풀 7개 ("좋은 하루 되세요!", "안녕하세요!", 등)
- 에셋: `assets/mascot/silver_dog.png` (fallback: `Icons.pets`)

### 3.2 JobCard 레이아웃

```
Row: IntensityPill (좌) + Salary (우)
Title: 24pt w800, "모집" 접미사 제거
Company: 17pt w400, letterSpacing 0.2
Row: GrayChip(구직형태) + GrayChip(도보거리) + GrayChip(D-day)
```

### 3.3 IntensityPill

| physicalIntensity | 문장 | 색상 | 배경 |
|---|---|---|---|
| `light` | `앉아서 일해요` | `#4CAF50` | `rgba(76,175,80,0.08)` |
| `moderate` | `서서 근무해요` | `#FF9800` | `rgba(255,152,0,0.08)` |
| `heavy` | `무거운 짐 있어요` | `#F44336` | `rgba(244,67,54,0.08)` |

- 테두리: 1.5px solid, borderRadius 10dp, padding 7h/12w
- font: 14pt, FontWeight.w700

### 3.4 GrayChip

- background: `AppColors.background`, border: `AppColors.divider` 1px
- borderRadius 8dp, padding 6h/10w
- font: 13pt, FontWeight.w500, `AppColors.textSecondary`

### 3.5 제목 "모집" 제거

```dart
String get displayTitle => title.replaceAll(RegExp(r'모집$'), '').trim();
```

### 3.6 FilterBar

- 가로 스크롤 칩 리스트 (ListView horizontal)
- 칩 높이 36dp, 수평 패딩 14dp, borderRadius 20dp
- 선택: primary 배경 + 흰색 텍스트
- 미선택: background 배경 + divider 테두리

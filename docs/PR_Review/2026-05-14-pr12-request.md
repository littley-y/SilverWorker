## 1. 개요

공고 상세 화면(JobDetailScreen)의 UI를 전면 개편하고, 지원현황 페이지와의 연동을 개선하며, 시니어 가독성을 위해 글자 크기를 고정합니다. 또한 목업 데이터의 회사명을 실제 기업명으로 교체하고, 마스코트를 은일이로 교체합니다.

- **공고 상세 카드 그리드**: 2×2 정보 카드 + 가로 상세 카드 + 업무 강도 상세 카드
- **지원 취소 플로우**: 지원현황 → 공고 상세 → 지원 취소 버튼
- **글자 크기 고정**: 130%로 고정, 설정에서 조정 불가
- **목업 데이터 정리**: underscore 제거, 실제 기업명 사용
- **마스코트 교체**: 은일이(강아지) + 애니메이션 추가

## 3. 구현 상세

### 3.1 JobDetailScreen 레이아웃

```
Header: 제목(모집 제거) + 회사명 + 급여
Divider
2×2 카드 그리드
  ├─ 근무시간 (access_time) → workHoursPerDay시간
  ├─ 근무기간 (calendar_today) → workPeriod
  ├─ 근무요일 (date_range) → workDays
  └─ 고용형태 (work_outline) → employmentTypeLabel
업무 세부 내용 카드 (가로)
자격 요건 카드 (가로)
업무 강도 상세 카드 (가로)
  ├─ 서있는 시간 → standing/sitting 여부
  ├─ 무거운 짐 → heavy_lifting 여부
  └─ 실내 / 외 → outdoor 여부
Bottom: 지원하기(파랑) or 지원 취소(빨강)
```

### 3.2 지원 취소 플로우

1. 지원현황 카드 탭 → `context.push(/job/{jobId})`
2. JobDetailScreen에서 `hasAppliedProvider`로 지원 여부 확인
3. 지원한 경우 → "지원 취소" 빨간 버튼
4. 탭 → 확인 다이얼로그 → `cancelApplication()` → status 'cancelled'

### 3.3 글자 크기 고정

- `FontSizeNotifier`: `fixedScale = 1.3`, `minScale = 1.3`, `maxScale = 1.3`
- `_load()`: 항상 1.3으로 설정, 저장된 값 무시
- `setScale()`: 고정값 유지, 실제 변경 불가
- SettingsScreen: Slider 제거, "130%" 정적으로 표시

### 3.4 목업 데이터

- 회사명: `빌딩_경비원_1_관리사무소` → `종로타워 관리사무소`
- 제목: `빌딩 경비원 모집` → `빌딩 경비원` (모집 제거)
- `workHoursPerDay`: 8 (대부분의 목업)

### 3.5 마스코트 은일이

- 이미지: `assets/mascot/silver_dog.png` (강아지 은일이)
- 둥둥 떠다니는 애니메이션 (2초 주기, 위아래 8%)
- 터치 시 좌우로 흔들리는 애니메이션 (wobble)
- 메시지 풀에 은일이 관련 메시지 2개 추가

## 3. 구현 상세

### 3.1 MascotWidget

- 크기: 기본 80×80dp, `size` 파라미터 조절 가능
- `Image.asset('assets/mascot/silver_bunny.png')` 사용, `BoxFit.contain`
- 에러 시 fallback: 회색 컨테이너 + `Icons.pets`
- `AnimatedScale`로 부드러운 등장 애니메이션 (선택적, `animated` 파라미터)

### 3.2 FontSizeNotifier

- 상태: `double fontScale` (기본 1.0, 범위 0.8 ~ 1.4)
- `SharedPreferences`로 설정 영구 저장 (키: `font_scale`)
- `setScale()` 시 `clamp(minScale, maxScale)` 적용
- `_load()` Future를 `initialized` getter로 노출하여 테스트 가능하도록 설계

### 3.3 SettingsScreen

- 상단: "설정" 타이틀 + 마스코트
- 글자 크기 섹션: Slider (0.8 ~ 1.4, 6 divisions)
  - 좌측 "작게" / 우측 "크게" 레이블
  - 현재 값 퍼센트 표시 ("100%", "120%" 등)
  - 실시간 미리보기 영역 (샘플 문구)
- 앱 정보 섹션: 앱 이름, 버전
- 설정 초기화 버튼: AlertDialog 확인 후 기본값(1.0) 복원

### 3.4 앱 전체 폰트 적용

`main.dart`의 `MyApp` → `MediaQuery`로 `textScaler: TextScaler.linear(fontScale)` 오버라이드:
```dart
MediaQuery(
  data: MediaQuery.of(context).copyWith(
    textScaler: TextScaler.linear(fontScale),
  ),
  child: MaterialApp.router(...),
)
```

## Spec 12: 홈 화면 UI 개선

### 개요
홈 화면(JobListScreen) 및 공고 카드(JobCard)의 UI/UX를 개선하여 시니어 사용자의 가독성과 정보 밀도를 높이고, 마스코트를 통한 감성 요소를 추가했습니다.

### 변경사항

1. **마스코트 배너 신규** (`lib/widgets/mascot_banner.dart`)
   - 화면 최상단에 배치, 스크롤 시 함께 날라감
   - 강아지 캐릭터 (🐶), 터치 시 랜덤 응원 메시지 7종 중 하나 출력
   - 말풍선 스타일 (흰색 배경, primary 테두리, 좌하단 꼬리)
   - `TweenAnimationBuilder` 등장 애니메이션

2. **JobCard 전면 개편** (`lib/widgets/job_card.dart`)
   - 상단: IntensityPill (좌) + 급여 (우)
   - 제목: 24pt w800, "모집" 접미사 제거 (`RegExp(r'모집$')`)
   - 회사명: 17pt w400, letterSpacing 0.2
   - 하단: GrayChip 3개 (구직형태 · 도보거리 · D-day) 한 줄 배치
   - 카드 크기 확대 → 한 화면에 2개 노출

3. **신규 위젯**
   - `IntensityPill`: 업무 강도 문장 표현 + 테두리 pill (초록/주황/빨강)
   - `GrayChip`: 연회색 배경 메타 정보 chip

4. **JobListScreen 레이아웃 변경**
   - `MascotBanner` 추가, AppBar actions에서 마스코트 제거
   - `_EmptyState`에 `SingleChildScrollView` 추가 (overflow 방지)

5. **JobModel 확장**
   - `walkingMinutes` 필드 추가 (목업용 도보 거리)
